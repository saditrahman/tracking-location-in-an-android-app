## What is the app about?

This is a very simple Expense Tracker application built using Java and Firebase. It is designed to run on Android OS mobile phones only.


## Technologies behind this:

<ol>
<li>Java</li>
<li>Google FusedLocationProvider API</li>
</ol>

## What are the features in it:

<ol>
<li>Get Latitude and Longitude</li>
<li>Get Altitude</li>
<li>Get Accuracy and Speed (if possible)</li>
<li>Get geo location</li>
</ol>
